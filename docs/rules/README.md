# Rules

## 목표

`docs/rules`는 AI 에이전트가 설계, 구현, 리뷰, 검증에서 반드시 따라야 하는 현재 규칙 문서의 홈이다.

이 디렉토리는 작업 산출물, TDD, API spec, 회고, 참고 문서를 보관하지 않는다.

## 문서 맵

| 영역 | 진입점 | 책임 |
|------|--------|------|
| Backend rules | [backend/README.md](backend/README.md) | 백엔드 아키텍처, 패키지, 코딩, 테스트, API, 데이터, 보안, 운영 규칙 |

## 규칙

- RULES-001: 규칙 문서는 `docs/rules/{area}` 하위에 둔다.
- RULES-002: 산출물과 참고 문서는 `docs/rules` 밖에 둔다.
- RULES-003: 새 규칙 영역을 추가하면 이 README의 문서 맵을 갱신한다.
- RULES-004: 규칙 문서와 산출물 문서가 충돌하면 규칙 문서를 우선한다.
