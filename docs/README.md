# Docs

## 목표

`docs`는 프로젝트 요구사항, 규칙, 설계 산출물, 운영·참고 문서의 최상위 지도다.

이 문서는 하위 문서의 상세 목록을 모두 소유하지 않고, 각 영역의 진입점만 연결한다.

## 문서 맵

| 영역 | 진입점 | 책임 |
|------|--------|------|
| Rules | [rules/README.md](rules/README.md) | AI 에이전트가 따라야 하는 규칙 문서 지도 |
| Backend artifacts | [backend/README.md](backend/README.md) | 백엔드 TDD, API spec, ADR 같은 산출물 지도 |

## 소유권

- `docs/README.md`는 docs 최상위 진입점만 소유한다.
- `docs/rules/README.md`는 규칙 문서 전체 지도를 소유한다.
- `docs/rules/backend/README.md`는 backend 규칙 Source of Truth를 소유한다.
- `docs/backend/README.md`는 backend 산출물 지도를 소유한다.
